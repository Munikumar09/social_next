Logo settings

Learn how to change logo and logo height

Logo can be changed from below code inside the src/components/LogoBox.tsx file.

<!-- Logo START -->
<Link class="navbar-brand" to="/">
  <img class="light-mode-item navbar-brand-item" src={logo} alt="logo">
</Link>
<!-- Logo END -->
Logo height

To change logo height, You should change the value in $navbar-brand-height in _variables.scss file as per your need.
